Grid Column Offsets

You can use .col-<bp>-push-<number> to push and pull columns. This can be used to center a layout, e.g. a 10/12 centered layout, or to create an overlapping magazine-style layout.

Creating a 10/12 centered layout is easier without using offsets, you can add the class row--center, e.g. row row--center like the font size section above.

You can see the columns layout with the Base Layers Chrome extension . 

col-xs-12 col-md-3 col-md-push-1
col-xs-12 col-md-4 col-md-pull-1
col-xs-12 col-md-4 col-md-pull-1
col-xs-12 col-md-3 col-md-pull-2
